Update the PDF after modifying it in an external viewer?

I love the import legacy annotations and highlights feature. I can open my files several times in external viewers (since I don't always work in computers where I may install Qiqqa), highlight them (or create new notes) and them import those changes into Qiqqa annotations. The problem is that, the next time I have to download and open the files in external viewers, in different (public computers), the files are the original files, without any annotations.
It would be most useful if we could have the option to replace the old PDF file with the new file with more bookmarks/annotations/highlights.
