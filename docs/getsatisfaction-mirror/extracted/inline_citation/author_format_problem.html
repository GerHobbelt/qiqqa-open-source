Inline citation - author format problem

Thanks for treating as urgent please, as I'm doing final formatting for my thesis, which I have to submit in two weeks. BTW: I've really enjoyed Qiqqa the last 3 years, and haven't met a blocker like this.

The problem is with inline citations, and I'm using the latest version of Qiqqa.
I keep getting the inline citation as (John Smith, 2013) instead of (Smith, 2003).
I've switched citation style so many times because I thought that was the problem (tried APA, APSA, Harvard and a few others)
Also, I've tried renaming the bibtex author entry - tried "John Smith", "Smith, John" and "Smith J" - all do not give me the change I want. I get (J Smith, 2013) with the later.
I'm really wondering what the problem is. Please advice.

Thanks for your help.

    Its now solved with some csl editing.
    Qiqqa support suggested that it may be an issue with 'disambiguation'

    There's lots of discussion on the problem here: https://forums.zotero.org/discussion/...

    Essentially I edited the csl by changing disambiguate-add-givenname="true" to disambiguate-add-givenname=""
