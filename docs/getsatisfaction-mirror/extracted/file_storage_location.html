File Storage Location

Greetings!

When I import many pdfs into a local library, and the pdfs are stored there, where is this material stored on my hard disk drive?

    You can also instruct Qiqqa to use a different directory as its data directory if you like.

    But beware of putting it under an auto-syncing folder (like DropBox, or GoogleDrive) because they can corrupt the database if you change the database on two computers simultaneously.

    Qiqqa (Official Rep) over 8 years ago
    Under the config screen there is a text box to override the location where you want Qiqqa to create its data store. If you are willing to start again, that is the easiest. Just enter the location you want to use for the Qiqqa database, and restart Qiqqa.

    If you already have a library, copy the contents of the old location to the new location, then enter the new location name into the text box, then restart qiqqa. Once you see that Qiqqa is using the new location, you can safely rename/move/delete the contents of the old location.
