    TJ over 8 years ago
    Ok I've solved my problem but I had to go all the way back to v43 to do it.

    Just in case anyone else has the same issue, just keep re-installing down the list of different versions until you find one that works.

    To do this, type the line below into your browser, replacing the ## with the version number you need:

    qiqqa.com/content/client/setup##.exe

    e.g. qiqqa.com/content/client/setup43.exe

    Hope this helps others with the same problem and that Qiqqa sorts out the issues with these later versions.
