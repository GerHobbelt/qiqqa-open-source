    • Dennis 1 year ago
      See my comments above from years ago.

      Qiqqa has some features that remain unmatched by any other software. Its ability to find themes / autotag and suggest the most important papers for review can be uncanny (especially when the AI is buried in closed source).

      But Qiqqa is showing its age:

      The Mozilla components used for the built in browser and sniffer are quite dated and even deprecated. This becomes an increasingly dangerous security hole.

      The CSL (citation style language) engine is many updates behind leaving us with non working styles as new features are added.

      Sniffer seems to be having problems keeping up with evolving Google Scholar and Pubmed interfaces.

      Identified bugs remain unaddressed.

      Jimme's announced intent is to run the Qiqqa money machine until it grinds to a halt. Without some maintenance, that day is rapidly approaching. I think I see the "check engine light" flashing.
