Android app version of Qiqqa for tablet

I am a new Qiqqa user. I am really impressed with the features in Qiqqa so far. One thing that would make Qiqqa much more versatile would be to add (at least) the following two capabilities to the Android app: 1) The ability to highlight text in yellow in the app and be able to sync it with the cloud (Qiqqa server) and the desktop version. Unless I am missing something, it seems like it is only possible to add and sync annotations in the form of text boxes. 2) The ability to view and edit a paper's metadata (author, title, journal, year, etc.) by hand, and to use the bibtex sniffer in the Android app on the tablet to look up and apply this data to each paper entry. Currently it is only possible to view and edit the comments and tags sections of the metadata in the Android app.
Those two improvements would make Qiqqa a lot more valuable because it would enable more productivity on the go, or during the daily commute. (And it would also make managing the bibtex metadata less of a pain.) It would be worth paying for the Premium version of Qiqqa if those features - that are already in the Desktop version - are added to the Android mobile app (and keep the mobile app free).
