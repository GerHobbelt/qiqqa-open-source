    Dez over 7 years ago
    i'm hoping i'm wrong here, but the only way I've found is to right-click on the tag in the tag list and hit "rename/delete tag" then leave it blank to delete. Which ain't terrible. Make sure you fully close Qiqqa down and re-open before expecting to see changes.
    The annoying bit is that this is only a PROXY to the actual auto-tags list, via the tags list. They don't sync, irritatingly. So if you've got more than a few autotags which aren't assigned to any paper, my cruddy workaround is to
    1. Pick a paper at random & note down its existing tags which you want to keep. I suggest Aaron A. Aaronson's "Aardvark Analysis", as this will be at the top, thus save scrolling.
    2. Go through the autotags by typing "a" and selecting all the duff ones that come up. Then do the rest of the vowels, and "y" if you're likely to have unwanted tags about rhythm, fly, spy, and other non-vowel words.
    3. Restart Qiqqa
    4. Go through the tags list in the library and right-click delete the duff tags
    5. Check Aaronson to ensure only the original tags remain
    6. Restart Qiqqa
    7. Be frustrated.
